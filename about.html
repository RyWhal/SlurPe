What is it?

Simple, you can take a URL like this: http://www.amazon.com/gp/product/B00AWH595M/ref=amb_link_385099882_2/189-5057054-9429847?ie=UTF8&nav_sdd=aps&pf_rd_m=ATVPDKIKX0DER&pf_rd_s=center-1&pf_rd_r=1ZPS578NXV53N34JCG45&pf_rd_t=101&pf_rd_p=1630880162&pf_rd_i=507846

And shorten it into something like this:
slur.pe/kindle


About the service:

SlurPE is a no frills URL shortener, owned and operated by Ryan Whalen. It was originally created as a weekend project in October of 2013. From there it become a frequent side project, taking up a good poriton of my free time. I will continue to improve and grow the service as time permits, however this is being created entirely in my spare time. If you have any feedback, comments, or suggestions, please feel free to email me: 'ryan@slur.pe'.


Features coming soon(ish?):

-Session cookies to track link's you've submitted without logging in (maybe).
-UI and code updates (performance matters!)
-QR codes to copy links to mobile devices
-Stats and Infographics!
